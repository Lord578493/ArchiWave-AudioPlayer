# 🎧 ArchiWave Audio Player (v1.0)

**ArchiWave** — это стильный, легковесный аудиоплеер на C#/WPF, созданный с упором на производительность и визуальную эстетику.

## ✨ Ключевые особенности
* **Оптимизация:** Нулевые аллокации памяти при отрисовке спектра (идеально для слабых ПК).
* **Визуал:** Неоново-синий интерфейс с динамическим FFT-визуализатором.
* **Скорость:** Мгновенное сканирование треков (до 500 файлов за пару секунд).
* **Удобство:** Поддержка Drag-and-Drop, сохранение плейлиста в JSON.

## 💻 Системные требования
* **ОС:** Windows 10/11
* **Процессор:** Тестировалось и отлично работает на Intel Core 2 Quad Q6600.
* **Архитектура:** x64
